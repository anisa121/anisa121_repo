Two Powerful Solutions in One App

Pathway 1: Peer Learning Orchestration

Core Hypothesis: Students learn best from peers just slightly ahead of them. Our classroom-based platform intelligently pairs students for structured micro-teaching sessions during supervised class time.

  • Assesses mastery by topic, not overall level
  • Creates optimal learning pairs (1-2 level gaps)
  • Provides 10-minute structured protocols
  • Gamified teaching practice with progress tracking

Pathway 2: Real-Time Engagement Thermometer

Core Hypothesis: Teachers can't differentiate without knowing engagement levels. A nominative real-time feedback system provides actionable data for immediate intervention.

  • Simple student interface: "Lost" / "Need example" / "Got it"
  • Pace feedback: "Too fast" / "Just right" / "Too slow"
  • Teacher dashboard with real-time, nominative data
  • AI-suggested interventions and exercise bank
